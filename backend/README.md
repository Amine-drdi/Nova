# Backend - Nova

API backend pour la plateforme Nova utilisant Hono, tRPC et Drizzle ORM.

## 🎯 Technologies

- **Hono 4.8** - Web framework
- **tRPC 11.8** - Type-safe API
- **Drizzle ORM 0.45** - Database ORM
- **MySQL 8** - Database
- **TypeScript 5.9** - Type safety

## 🚀 Installation

```bash
cd backend
npm install
```

## ⚙️ Configuration

```bash
cp .env.example .env
# Modifier .env avec vos valeurs
```

## 🔧 Scripts

```bash
npm run dev      # Development server (port 3000)
npm run build    # Build for production
npm start        # Start production server
npm run lint     # Linting
npm run format   # Format code
```
